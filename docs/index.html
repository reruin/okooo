OKOOO Robot API

Last updated May. 21, 2015

本文档是 OKOOO 抓取接口文档。

Soccer

参数表

参数 必填 描述
id 是 赛事id
type 否 获取数据类型.
百家欧指：BaijiaBooks，
权威：AuthoriteBooks，
大公司：BigBooks

返回结果

字段 类型 描述
MakerID int 公司ID
CompanyName string 公司名
CompanyCountry string 公司所属国家
Start object 初指
Start.home number 主
Start.draw number 平
Start.away number 客
End object 最新指数
End.home number 主
End.draw number 平
End.away number 客
Radio object 概率
Radio.home number 主
Radio.draw number 平
Radio.away number 客
Kelly object 最新凯利指数
Kelly.home number 主
Kelly.draw number 平
Kelly.away number 客
Payoff number 赔付率
startUpdatetime number 最初更新时间 与 当前的时间差，单位秒
Updatetime number 最近更新时间 与 当前的时间差，单位秒
spchangeHometeam array 分时数据
spchangeHometeam[].Change array 赔率[主,客,平]
spchangeHometeam[].kelly array 凯利值[主,客,平]
