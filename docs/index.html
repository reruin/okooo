OKOOO Robot API

Last updated May. 21, 2015

本文档是 OKOOO 抓取接口文档。

收益计算模型

/okooo/api/?m=soccer&a=calc&model={%22calc%22:[1,2],%22bet%22:%22williamhill%22,%22odds%22:[{%22home%22:[0,1.25],%22away%22:[5,11],%22draw%22:[6,9.5],%22type%22:%22williamhill%22},{%22home%22:[0,1.3],%22type%22:%22oddset%22}]}

参数 model
格式
    {
        // 胜负平下注, 0 负，1平，2胜
        "calc":[1,2],
        // 在哪家下注
        "bet":"williamhill",
        // 模型
        "odds":[
            {
            "home":[0,1.25], //胜赔区间
            "away":[5,11], // 负赔区间 , 如若全部，不赋值即可
            "draw":[6,9.5], // 平赔区间
            "type":"williamhill" //数据来源
            },
            {
                "home":[0,1.3], //胜赔区间
                "type":"oddset" //数据来源
            }
        ]
    }

williamhill初赔数据中，胜赔0 ~ 1.25间,负赔 5 ~ 11间，平赔 6~9.5间
且
oddset初赔数据中，胜赔1.3 以内
的所有比赛，按 williamhill 的赔率，下注 平胜，计算此种方式的收益。

返回：
{"status":"0","detail":{"count":57,"r":-0.1278}}
count：满足此模型的比赛数量。
r：此模型的收益率，1表示 100%

Soccer

参数表

参数 必填 描述
id 是 赛事id
type 否 获取数据类型.
百家欧指：BaijiaBooks，
权威：AuthoriteBooks，
大公司：BigBooks

返回结果

字段 类型 描述
MakerID int 公司ID
CompanyName string 公司名
CompanyCountry string 公司所属国家
Start object 初指
Start.home number 主
Start.draw number 平
Start.away number 客
End object 最新指数
End.home number 主
End.draw number 平
End.away number 客
Radio object 概率
Radio.home number 主
Radio.draw number 平
Radio.away number 客
Kelly object 最新凯利指数
Kelly.home number 主
Kelly.draw number 平
Kelly.away number 客
Payoff number 赔付率
startUpdatetime number 最初更新时间 与 当前的时间差，单位秒
Updatetime number 最近更新时间 与 当前的时间差，单位秒
spchangeHometeam array 分时数据
spchangeHometeam[].Change array 赔率[主,客,平]
spchangeHometeam[].kelly array 凯利值[主,客,平]
